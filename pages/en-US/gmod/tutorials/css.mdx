# 🟪 Missing CSS content
<Callout type="info" emoji="ℹ️">
    [Here's a video that you can use if you don't want to read bellow steps.](https://youtu.be/8AfuNpX8RQA)
</Callout>

## 1. Without Counter-Strike Source game
### Requirements
- [ ] Garry's Mod must be **installed**
- [ ] [WinRAR](https://www.win-rar.com/start.html?&L=10) or any software that can **extract ZIP / RAR archives**

### Tutorial
1. Find a website that **offers CSS content download** *(we recommend [this one](https://gmodcontent.com/))*
2. **Extract the ZIP / RAR folder into a new folder** named ``csscontent`` for example
<FileTree>
  <FileTree.Folder name="csscontent" defaultOpen>
    <FileTree.File name="materials" />
    <FileTree.File name="models" />
    <FileTree.File name="particles" />
    <FileTree.File name="sound" />
  </FileTree.Folder>
</FileTree>
3. **Find your Garry's Mod addon's folder** on your computer and **move** ``csscontent`` folder in it
4. **Restart / Start your game**, content should be properly mounted

## 2. With Counter-Strike Source game
<Callout type="info" emoji="ℹ️">
    Garry's Mod should **automatically mount CSS content** if it detects that **Counter-Strike Source is installed**.  
    The following steps can be used instead to **manually mount it**.
</Callout>

### Requirements
- [ ] Garry's Mod and Counter-Strike Source **should be installed**
- [ ] Garry's Mod must be able to **access Counter-Strike Source folder**

### Tutorial
1. **Start Garry's Mod**
2. In **main menu**, hit **🎮 Games button** in bottom right
3. **Verify** that Counter-Strike Source is ✅, **check it** if not
4. **CSS content should be mounted!**
